# Java First Project

The classic "Hello, World!" program — the very first step in learning Java.

## What It Does

Prints `Hello, World!` to the console using `System.out.println`.

## Project Structure

```
FirstProject/
└── src/
    └── FirstProject.java   # Main class with the entry point
```

## Running the Project

Open `FirstProject/` in IntelliJ IDEA and run `FirstProject.java` as the main class.

Alternatively, compile and run from the command line:

```bash
javac src/FirstProject.java -d out/
java -cp out/ FirstProject
```

## Requirements

- Java 8+
- IntelliJ IDEA (or any Java IDE / `javac` on the command line)
